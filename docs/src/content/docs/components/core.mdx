title: Core
---

Common functions and systems used by other mods.

Additional features:

- Turret radial menu
- More gestures (_e.g._ sit on the floor)

### Usage

#### Turret Inspection Radial Menu

- Use the `Inspect` keybind in `Equipment` category (Default: Hold <kbd>R</kbd>)

#### Play Gestures

- Use the `Open gestures menu` keybind in `Character` category (Default: <kbd>F1</kbd>)
