title: Trenches
---

ACE Trenches extends the E-tool with the ability to place dirt covers.

## Usage

### Place Dirt Cover

- Toggle building mode while having the E-tool in hand with the `Use item` keybind in `Equipment` category (Default: <kbd>R</kbd>).
- Place dirt cover while in building mode with <kbd>LMB</kbd>.
- Switch dirt cover type while in building mode with <kbd>V</kbd>.
