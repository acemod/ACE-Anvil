ACE Facepaint adds several camouflage facepaint items which apply or remove faction-specific facepaint.

## Usage

### Apply Facepaint

- Put facepaint in hand and use the self-interaction keybind (as if applying a bandage).

### Remove Facepaint

- After applying, put facepaint in hand and use the self-interaction keybind (as if applying a bandage).
