This adds the option to carry or drag incapacitated units and load them in vehicles.

## Usage

### Carrying incapacitated units

- The torso will show a user action for `Carry` and `Drag`
- The carried unit can be released with `Release` keybind in `ACE` category (Default: <kbd>Ctrl</kbd>+<kbd>X</kbd>).

### Loading incapacitated units

- When carrying a unit, you can place them in vehicles with the user action on the door.
